# pdfdancer-fonts

Font collection used by [PDFDancer](https://www.pdfdancer.com) for font substitution and matching during PDF editing. When a PDF references a font that isn't embedded, PDFDancer's ML-driven font intelligence uses this collection to find a suitable replacement.

## Setup

Requires [Git LFS](https://git-lfs.github.com/) — all `.ttf` files are stored via LFS.

```
git lfs install
git clone git@github.com:MenschMachine/pdfdancer-fonts.git
```
